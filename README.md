## What is this ??
 This is simple todo lists.

## How to use ??

### Step 1
- Clone project with SSH or HTTPS 

### Step 2

```
composer install
```

### Step 3

```
cp .env.example .env
```
Then, config your database.


### Step 4
```
php artisan key:generate
```
### Step 5

```
php artisan migrate
```

### Step 6
- Simple run use:

```
php artisan serve
```

- Crud basic for todolist
- Pagination
